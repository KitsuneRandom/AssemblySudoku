# AssemblySudoku
Résolveur de sudoku en assembleur

Comment l'utiliser ? ce résolveur de sudoku ne prend pas une grille a résoudre, mais juste les 81 chiffres qui compsent cette grille (mettre des 0 a la place des trous). il suffit de les placer dans sudoku.txt et de changer le chemin d'acès au fichier dans le sudoku.asm, au niveau du .data (mettre le chemin absolu). 

il affichera en valeur de retour une grille avec une solution possible
